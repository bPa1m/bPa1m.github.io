# FaceToFace - GGJ 2026

FaceToFace is a game about taking care of a creature which communicates its emotions using pre-made masks. Your goal is to finish the check list left by your supervisors without upsetting the creature

---


This game was developed as a part of Global Game jam 2026, you can find it [here](https://globalgamejam.org/games/2026/face-face-7)

---

### Game Features:
   - Developed using **Godot**
   - Dynamic mood system
   - Creature and object **models**
   - Creature **animations**
